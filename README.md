# JeuxdeBallon

# Pré-requis<br/>

Cmake >=2.8. <br/>


# Code éxecution<br/>
1. Sur terminal, éxecutez commande *cmake* <br/>
2. Puis, éxecutez commande *make* <br/>
3. Puis, éxecutez le projet avec commande *./JeuxdeBallon* <br/>
